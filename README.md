## Pre-requisites
- **+18.09** docker
- **+1.23** docker-compose

## Installation
1. open the CMD on the root folder of the project
2. execute `docker-compose up -d`
3. execute `docker exec not-a-bot composer install`
4. execute `docker exec not-a-bot php ./entrypoint.php`
